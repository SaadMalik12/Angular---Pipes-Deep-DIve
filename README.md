````md
# Pipes Deep Dive

An Angular project built to practice and demonstrate Angular Pipes, including built-in pipes, custom pipes, pure vs impure pipes, and real-world data formatting examples.

## Topics Covered

- What are Pipes?
- Built-in Pipes
- Custom Pipes
- Pure vs Impure Pipes
- Temperature Conversion
- Date Formatting
- Data Transformation in Templates

## Features

- Real-time temperature formatting
- Historic temperature display
- Custom Angular pipes
- Clean responsive UI
- Practical examples of data transformation

## Tech Stack

- Angular
- TypeScript
- HTML
- CSS

## Live Demo

🚀 https://angular-pipes-deep-d-ive.vercel.app/

## Getting Started

Clone the repository:

```bash
git clone https://github.com/your-username/your-repository-name.git
````


Run the development server:

```bash
ng serve
```

Navigate to:

```bash
http://localhost:4200/
```

## Purpose

This project was created for learning and practicing Angular Pipes and understanding how data transformation works inside Angular templates.
